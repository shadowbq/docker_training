# NFS Volume mounts

Sometimes we want to have the persistant storage on a NAS or some other like system. Rancher has a NFS Storage Driver that can mount shares from a running NFS Server.

* http://rancher.com/docs/rancher/latest/en/rancher-services/storage-service/
* http://rancher.com/docs/rancher/v1.6/en/rancher-services/storage-service/rancher-nfs/
